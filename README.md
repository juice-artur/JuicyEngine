# Goals

---
# Project setup

## Cloning
```bash
git clone https://github.com/juice-artur/JuicyEngine.git --recursive
```

## Build
1. Create a build directory in the project root: ```mkdir build && cd build```
2. Generate project files with CMake: ```cmake ..```
